## Project Overview
This project performs exploratory data analysis (EDA) on Netflix's content dataset to uncover trends in content growth, genre popularity, and country distribution.

## Objectives
- Analyze Movies vs TV Shows distribution
- Identify content growth trends over time
- Explore top producing countries
- Perform genre-level analysis using multi-label splitting

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Insights
- Netflix saw rapid content expansion between 2016 and 2019.
- International Movies dominate the platform.
- United States produces the highest number of titles.
- Drama and Comedy are among the most popular genres.

## Skills Demonstrated
- Data Cleaning
- Handling Missing Values
- Datetime Conversion
- Feature Engineering
- Multi-label Transformation using explode()
- GroupBy Aggregation
- Data Visualization
- Business Insight Interpretation
